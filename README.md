# gz-project

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
# 使用方式:
```js
 <vue-count-up :end="2500" :duration="2.5"></vue-count-up>
```
